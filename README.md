## 2. 업무 진행
### 디자이너: 요구사항에 맞도록 디자인 한 결과물을 웹 퍼블리셔에게 전달
### 웹 퍼블리셔: 이 디자인을 기반으로 HTML, CSS를 만들어 개발자에게 제공
### 백엔드 개발자: HTML 화면이 나오기 전까지, 시스템을 설계하고 핵심 비즈니스 모델을 개발. (이후 HTML이 나오면, 이 HTML을 뷰 템플릿으로 변환해서 동적으로 화면을 그리고, 웹 화면의 흐름을 제어)
